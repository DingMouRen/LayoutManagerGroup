### Just like a star.

**Gradle import:**

1. Add in the Root level `build.gradle`:

```
	allprojects{

		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

2. Add Dependency:
```
dependencies {
	     implementation 'com.github.DingMouRen:LayoutManagerGroup:1e6f4f96eb'
	}
```
